About me

I am a proactive financial sector professional with a strong interest towards data science. In my day-to-day work I build credit risk models for Nordic banks using mainly R. This website is my personal initiative to refamiliarize myself with the possibilities of modern website development.

I still remember when me and my friends used to learn to code in the beginning of the 21st century writing raw HTML code and using sophisticated tools like notepad and paint. I regarded it highly intriguing although the website creation was usually deemed of not having any sensible content to put on the site.

During the recent years, it has been fun to observe the big hype around the website and app creation as the concepts have, as seen through my still limited understanding, remained quite similar to the ones I learned 15 years ago. In this website, I've written mainly HTML together with an associated CSS file so I've concentrated on the front-end part. I hope to get a chance to familiarize myself with the vast amount of back-end technologies also in the future.
